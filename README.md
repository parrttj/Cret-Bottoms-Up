# Web-Based Day Planner Application

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone https://github.com/parrttj/Cret-Bottoms-Up.git
   cd Cret-Bottoms-Up
   ```

2. **Install dependencies**:
   Make sure you have Node.js installed. Run:
   ```bash
   npm install
   ```

3. **Build the application**:
   ```bash
   npm run build
   ```

## Opening Instructions

To start the application, run:
```bash
npm start
```

This will launch the application on `http://localhost:3000`.

## User Guide

- **Adding a new task**: Click the 'Add Task' button and enter the task details.
- **Viewing tasks**: Tasks are displayed in a daily view. You can scroll through the day to view tasks.
- **Editing a task**: Click on a task to edit its details.
- **Deleting a task**: Click the 'Delete' button on the task you wish to remove.
- **Marking tasks complete**: Click on the checkbox next to the task to mark it as complete.

## Conclusion

This application allows users to efficiently plan their day. Make sure to save your work regularly!
